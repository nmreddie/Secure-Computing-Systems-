# Secure Computing Systems – Security Tools Portfolio

## Overview
Three command-line security tools built in Python to demonstrate 
secure authentication, threat detection, and malware analysis.

---

## Tools

### 1. Secure Authentication System (`auth.py`)
Implements secure user login with SHA-256 hashing and account lockout.

**Run:**
```bash
python3 auth.py
```

---

### 2. SIEM Log Analysis Tool (`siem.py`)
Analyses system log files to detect brute-force attacks and suspicious IPs.

**Run:**
```bash
python3 siem.py auth.log 5
```

**Output:** Generates `siem_report_YYYYMMDD_HHMMSS.json`

---

### 3. Malware Analysis Tool (`malware_analysis.py`)
Scans files using SHA-256 signature matching and quarantines malicious files.

**Run:**
```bash
python3 malware_analysis.py
```

---
